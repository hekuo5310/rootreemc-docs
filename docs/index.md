# Rootree 文档系统

这是一个对标 VuePress 的极简静态文档系统，默认会将 `docs/` 下的 Markdown 文件自动渲染为 HTML 页面。

## 系统特性

- 自动扫描 Markdown 并生成页面路由
- 提供顶部导航、左侧文档结构与右侧页面目录
- 全站白底、黑色文字与黑色边框按钮
- 提供 `build` 与 `dev` 两个核心命令

## 快速开始

1. 运行 `npm run build` 生成静态站点
2. 输出目录为 `dist/`
3. 本地预览可运行 `npm run dev`
